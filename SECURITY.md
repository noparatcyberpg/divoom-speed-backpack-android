# Security Policy

## Reporting Security Issues
If you discover a security vulnerability or sensitive data exposure issue, please do **NOT** create a public issue. Email the repository maintainers or use private vulnerability reporting on GitHub.

## Data & Credentials Security
- No API keys, signing keystores, tokens, or private location credentials are checked into source control.
- All Bluetooth connection information and debug logs anonymize MAC addresses by default (`AA:BB:CC:**:**:FF`).
