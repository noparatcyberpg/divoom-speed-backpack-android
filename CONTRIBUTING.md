# Contributing to Divoom Speed Backpack

Thank you for contributing!

## How to Contribute
1. Fork the repository on GitHub.
2. Create a feature branch (`git checkout -b feature/my-new-feature`).
3. Ensure unit tests pass (`./gradlew testDebugUnitTest`) and lint checks pass (`./gradlew lintDebug`).
4. Commit your changes following standard commit prefixes (`feat:`, `fix:`, `chore:`, `test:`, `docs:`).
5. Open a Pull Request referencing your changes.

## Protocol Capture Submissions
If testing with a new Divoom device model, please use the **Bluetooth Inspector** tool to capture packet hex logs and submit a [Device Support Issue](.github/ISSUE_TEMPLATE/device_support.yml).
